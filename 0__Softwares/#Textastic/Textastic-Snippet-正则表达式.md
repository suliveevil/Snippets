# Textastic 正则表达式

## 基本原则

对象：已有文本、用户输入的文本、用户想要得到的目标文本

先匹配（match）已有文本+输入文本，如果匹配，则进行下一步，
再捕获（capture）输入文本，
将捕获的输入文本与string（代码片段的alias）进行对比，如果匹配，则进行下一步，
最后将输入文本替换为（replace）目标文本或在输入文本后面追加（append）目标文本
